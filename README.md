# Bangumi Data Helper

[![License](https://img.shields.io/npm/l/bangumi-data-helper.svg)](https://github.com/bangumi-data/helper/blob/master/LICENSE)

```bash
npm i bangumi-data-helper -g
```

需要在 `bangumi-data` 项目目录下运行

```bash
# 生成某一季度的初始数据
bdh create 2016q4
```
